## 프롬프트 관리

|  | 어드민 | 유저 |
| --- | --- | --- |
| 프롬프트 만들기 | O | O |

## 스페이스 관리

|  | 어드민 | 유저 |
| --- | --- | --- |
| 스페이스 테마 등록하기 | O | X |
| 유저, 그룹, 토큰 제한량 관리하기 | O | X |
| 통계 확인하기 | O | X |
| 예시 질문 등록하기 | O | X |

## 계정 관리

| --- | --- | --- |
