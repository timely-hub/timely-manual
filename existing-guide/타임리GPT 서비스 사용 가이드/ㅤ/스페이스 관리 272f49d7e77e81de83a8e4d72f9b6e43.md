# 스**페이스 관리**

![어드민설정.png](%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4%20%EA%B4%80%EB%A6%AC/%E1%84%8B%E1%85%A5%E1%84%83%E1%85%B3%E1%84%86%E1%85%B5%E1%86%AB%E1%84%89%E1%85%A5%E1%86%AF%E1%84%8C%E1%85%A5%E1%86%BC.png)

생성된 **스페이스의 이름, 로고, 추천 템플릿** 등을 관리할 수 있어요.

## 메인화면

![스크린샷 2026-03-03 095423.png](%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4%20%EA%B4%80%EB%A6%AC/%EC%8A%A4%ED%81%AC%EB%A6%B0%EC%83%B7_2026-03-03_095423.png)

스페이스 홈페이지 접속 시 처음 접하는 메인 화면을 관리하는 부분이에요.

사용 유무에 따라 불필요한 부분은 숨길 수 있어요.

- **메인 상단 로고** :  320 * 100픽셀 미만 / 최대 10M의 jpg, jpeg, png 이미지 파일
- **스페이스 이름** : 좌상단에 노출되는 명칭(최대 9자)
- **스페이스 소개글** : 배너 옆에 노출되는 문구(OFF시 배너가 크게 보여요)
- 배너 7:2 비율 / 최대 10M의 JPEG, PNG 이미지 파일
- 추천 템플릿 최대 24개 등록
