## 2. 크레딧 결제 취소

- 크레딧 결제 취소는 결제일로부터 14일 이내에만 가능하며, 구입한 크레딧을 사용하지 않은 경우에 한해 가능합니다.

![어드민변경 크레딧결제취소.png](%ED%81%AC%EB%A0%88%EB%94%A7%20%EA%B2%B0%EC%A0%9C/%E1%84%8B%E1%85%A5%E1%84%83%E1%85%B3%E1%84%86%E1%85%B5%E1%86%AB%E1%84%87%E1%85%A7%E1%86%AB%E1%84%80%E1%85%A7%E1%86%BC_%E1%84%8F%E1%85%B3%E1%84%85%E1%85%A6%E1%84%83%E1%85%B5%E1%86%BA%E1%84%80%E1%85%A7%E1%86%AF%E1%84%8C%E1%85%A6%E1%84%8E%E1%85%B1%E1%84%89%E1%85%A9.png)

1. [고객센터 문의] : 결제 수단이 가상계좌, 인터넷뱅킹, 수동지급일 경우.
2. [결제취소] : 결제 수단이 카드나 페이머니일 경우.
3. [취소불가] : 결제일로부터 14일이 경과했거나, 구매한 크레딧을 사용했을 경우.
4. [환불처리완료] : 환불 처리가 완료된 상태.
